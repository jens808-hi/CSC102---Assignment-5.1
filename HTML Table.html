In 1989 Timothy Berners-Lee among other researchers at CERN created hypertext, which enables users to navigate to separate and related documents by using key words to link those documents together. For this concept to work a "markup language" that allowed links to be added to text had to be created. HTML files are plain text files. The extension .html and the entire document is encased within html open and close tags.

HTML page is separated into two parts:

<.HEAD tags> Describes the page itself

<.TITLE tags> Positioned in between the HEAD tags and is displayed on the web browser tab; used by search engines and OS; signaling to the user what they are viewing.

<.BODY tags> Contains the information we display to the user. Javascript can be used here to output information to the user.

<.P tags> Used to wrap around each paragraph of text

Debugging is a method of identifying the problem (aka bugs), isolating it at the source, correcting the errors in the code, and then testing it. Debugging helps troubleshoot the root cause of coding errors, prevents software malfunctions and improves performance. Debugging is done when a developer finds coding errors in a program and is able to reproduce it. When attempting to identify bugs, an integrated development environment (IDE) is a very useful tool, highlighting lines of code corresponding to error messages simutaneously in real-time effectively pinpointing the root of the problem. Software Development refers to to a process where a series of tasks are dedicated to creating, designing, deploying, testing, maintaining, and supporting software applications. The function of these processes is to create programmable software to cater to the business or personal objectives and goals through writing computer code.

There are 6 stages to the Software Development Cycle (SDLC):
1. Planning
2. Analysis
3. Design
4. Implementation
5. Testing & Integration
6. Maintenance

Popular Software Development Processes are:

-Waterfall Development: Steps are executed linearly and sequentially in a waterfall approach.

-Agile Development: Focues on flexibility, continuously evolving functionality by an incremental approach.

-Rapid Application Development (RAD): Focuses prototyping on speed and flexibility over strict planning.

-Rational Unified Process (RUP): Framework that is tailored by organizations to best suit the specific needs of different projects.

Strings hold data that represent a sequence of characters, words or symbols as a single unit that are enlcosed in quotation marks. By changing specific parts of a string you can replace characters of a string. Converting a string to upper and lower case is an example of transforming strings. "Event driven programming is known as an event loop. The loop performs 3 tasks: gather inputs, process data, and outputs the results." Graphical User Interfacaes (GUI) programs like web-browsers are event driven programs that run a loop, checking for user input, ready to make changes dependent on the data that was recieved and then ouputs it on the screen. Event driven programming enabe developers to create real-time applications that are more flexible, innovative and responsive than traditional programs.

Common data structures in programming are:

-arrays: lists of related values identified by it's index number

1. parallel arrays - two or more arrays are used to represent a group of data where each corresponding array index is associated with a matching field for a specific record.
2. associative arrays - a data structure that uses strings as indexes to access the elements of an array instead of it's numerical index

-linked lists: linear collection of nodes, where each node contains data and points to the next node

-stacks: organized items by a last-in, first-out order; most recent item added to the stack will be the first removed

-trees: non-linear structure that holds data in a hierarchical way, utilizing nodes by links that connected in a tree

1. binary trees - common type of tree where each parent nodes has at most two children; the purpose of these types of search trees is to fulfill a certain ordering property
2. leaves - last nodes in a tree that bare no children

Object Oriented Programming (OOP) is a model for writing programs and programming languages based on abstraction of objects, through this process a programmer conceals the data and code except for what is relevant to an object in order to reduce complexity, making it easier to work with. This can contain data in fields known as attributes or properties and code in procedures known as methods. For instance a button class has variables for data and functions, which can be inherited by a subclass and created as a subset of the button class. Objects are instances of classes. Using a concept known as structure or struct we can store multiple variables in one name:

struct myDog {
    var breed;
    var size;
    var coat;
    var color;
    var name;
    var weight;
}

Common data structures continued...

-queues: linear data structure tha stores and manipulates the data as a First In First Out order. Consists of two ends; front and rear, where elements are added and removed, respectively.

1. Used to manage threads, implements priority queuing and is impartial.An example where this is used is in virutal messaging apps to maintain the order of text messages.

-heaps: a binary tree-based data structure; reserved area of memory in a programming language. The value of the parent node is less than or equal to the value of it's children

-graphs: non-linear data structure that represents a network of interconnected nodes connected by edges. Each edge signifies a relationship between two nodes. Representation of complex connections between different data points within a system like a map with cities and roads. Cities are to nodes as roads are to edges connecting them.

-hash tables: stores key-value pairs and uses a hash function to generate an index value for each key(data element) to retrieve the data

1. Hash tables provide faster access to data due to the index value acting as a key
2. Widely used in many kinds of computer software for its efficiency in comparison to search trees
